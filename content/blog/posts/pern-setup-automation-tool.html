PERN Setup: Cross-Platform Full-Stack Automation Tool

Published on January 25, 2025 by RosettaScripts Team

PERN Setup PERN Stack Automation Full-Stack Development

PERN Setup is a powerful, cross-platform boilerplate generator that creates modern PostgreSQL + Express.js + React + Node.js applications in minutes. This automation tool eliminates the tedious setup process, allowing developers to focus on building features rather than configuring environments.

What is PERN Setup?

PERN Setup is designed for full-stack developers, startups, students, and teams building modern web applications with the PERN stack. It provides a lightning-fast way to create production-ready applications with security best practices built-in and modern development tools configured out of the box.

Key Features

  • Lightning Fast: Create a complete PERN Stack app in under 2 minutes
  • Production Ready: Security best practices, JWT auth, rate limiting built-in
  • Cross-Platform: Works on Linux, macOS, Windows (PowerShell & WSL2)
  • 5 Professional Templates: From simple APIs to microservices architectures
  • Modern Dev Tools: TypeScript, ESLint, Prettier, Jest, Docker, CI/CD ready
  • Zero Learning Curve: Comprehensive docs and examples included

System Requirements & Compatibility

PERN Setup supports multiple operating systems and shell environments:

Operating System Compatibility

Operating System Shell Script PowerShell Script Recommended
Linux ✅ Fully Supported ❌ Not Available Bash Script
macOS ✅ Fully Supported ❌ Not Available Bash Script
Windows (WSL2) ✅ Fully Supported ❌ Not Available Bash Script
Windows (PowerShell) ❌ Not Supported ✅ Fully Supported PowerShell Script

Prerequisites

  • Node.js 16+ (recommended: 18 LTS)
  • npm 8+ or yarn 1.22+
  • Git 2.20+
  • PostgreSQL 12+ (optional, can be installed via script)

Getting Started

PERN Setup is incredibly easy to use across different platforms:

For Linux/macOS Users

# Clone the repository
git clone https://github.com/rosettascript/pern-setup.git
cd pern-setup

# Make the script executable
chmod +x run.sh

# Run the setup
./run.sh

For Windows Users

# Clone the repository
git clone https://github.com/rosettascript/pern-setup.git
cd pern-setup

# Run the PowerShell script
.\run.ps1

Project Structure

The generated PERN Setup project follows a well-organized structure:

pern-setup/
├── run.sh              # Main setup script (Linux/macOS)
├── run.ps1             # PowerShell script (Windows)
├── lib/                # Library functions
│   ├── utils.sh        # Utility functions
│   ├── detect.sh       # System detection
│   ├── install.sh      # Installation functions
│   ├── database.sh      # Database setup
│   ├── security.sh     # Security configuration
│   ├── devtools.sh     # Development tools
│   └── validator.sh     # Validation functions
├── templates/          # Project templates
│   ├── starter/        # Starter template
│   ├── fullstack/      # Full-stack template
│   ├── api-only/       # API-only template
│   └── microservices/  # Microservices template
└── docs/              # Documentation

Environment Variables

PERN Setup uses environment variables for configuration:

DB_PORT=5432
DB_NAME=myapp
DB_USER=myuser
DB_PASSWORD=mypassword

# Application Configuration
NODE_ENV=development
PORT=5000
JWT_SECRET=your-secret-key
CORS_ORIGIN=http://localhost:3000

Troubleshooting

Common issues and their solutions:

Permission Denied (Linux/macOS)

# Fix script permissions
chmod +x run.sh
chmod +x lib/*.sh

PowerShell Execution Policy (Windows)

# Set execution policy
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Or run with bypass
.\run.ps1 -ExecutionPolicy Bypass

Node.js Not Found

# Install Node.js via package manager
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# macOS (with Homebrew)
brew install node

# Windows (with Chocolatey)
choco install nodejs
